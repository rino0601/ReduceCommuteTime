Contribute
=============

이 장에서는 본 Repo의 개발에 참여하기 위해 알아야 할 지식에 대해서 설명 합니다.

Requirements
''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''

본 프로젝트 요구 환경은 아래와 같습니다.

- python3.6 or later. but, no python3.7(airflow doesn't support)
- docker 18.06.1-ce or later

개발시에는 local 환경에서 빌드 스크립트로 실행 가능 하고, 테스트는 docker-compose 를 통해 가능합니다.
배포처는 아직 결정 된게 없습니다.
